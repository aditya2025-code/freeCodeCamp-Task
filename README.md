# Web_Dev-Task

A collection of HTML practice exercises and daily coding tasks completed as part of learning web development through [freeCodeCamp](https://www.freecodecamp.org/).

## 📌 About

This repository is a personal learning log used to track progress on HTML fundamentals while working through freeCodeCamp's curriculum. Each task or exercise is added here as it's completed, making it easy to look back and see how things have progressed over time.

## 📁 Repository Structure

```
Web_Dev-Task/
├── Daily-Task/             # Day-by-day practice exercises
└── Web Dev-TASK/
    └── HTML/                # HTML challenge solutions
```

- **Daily-Task** — Smaller, day-to-day coding exercises used to reinforce concepts as they're learned.
- **Web Dev-TASK/HTML** — Solutions and practice files for HTML-focused challenges from the freeCodeCamp curriculum.

## 🛠️ Tech Stack

- **HTML** — currently the only language used in this repository.

## 🚀 Getting Started

No build tools or installation required. To view any exercise:

1. Clone the repository:
   ```bash
   git clone https://github.com/aditya2025-code/Web_Dev-Task.git
   ```
2. Navigate into the folder of the task you want to view:
   ```bash
   cd Web_Dev-Task/Daily-Task
   ```
3. Open the relevant `.html` file directly in your browser, or use a tool like the [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) extension in VS Code for live reloading while editing.

## 🎯 Purpose

This repo exists purely for learning and practice. It's a running record of progress through freeCodeCamp's lessons rather than a polished project — expect it to grow and change as more topics are covered.
